# Neha - Portfolio

A modern, responsive portfolio website featuring smooth scrolling, beautiful animations, and a periwinkle blue + black theme.

## Features

✨ **Modern Design**
- Black background with periwinkle blue accents
- Smooth scrolling navigation
- Responsive grid layouts
- Floating animations and transitions

🎨 **Customizable**
- Easy to update colors via CSS variables
- Modular sections (Hero, About, Projects, Skills, Contact)
- Space Flex typography

📱 **Mobile Friendly**
- Fully responsive design
- Mobile hamburger menu
- Touch-optimized interactions

⚡ **Performance**
- Lightweight and fast
- Optimized animations
- Smooth scroll behavior

## Getting Started

### Visit Your Portfolio
👉 **https://nehajatla.github.io/portfolio/**

### Customize Your Portfolio

1. **Update Personal Information**
   - Edit your email in the Contact section
   - Add your LinkedIn and GitHub URLs
   - Update project descriptions
   - Modify skill percentages

2. **Add Your Images**
   - Create `assets/images/` folder
   - Add: `profile.jpg`, `project1.png`, `project2.png`, `project3.png`

3. **Change Colors** (Optional)
   Edit CSS variables in `styles.css`:
   ```css
   :root {
       --primary: #7c5cfa;    /* Periwinkle blue */
       --dark-bg: #0a0a0a;    /* Black background */
   }
   ```
